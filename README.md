## 下载

### Docker

```
docker pull apachecn0/succinctly-zh-pt3
docker run -tid -p <port>:80 apachecn0/succinctly-zh-pt3
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install succinctly-zh-pt3
succinctly-zh-pt3 <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```

# 访问 http://localhost:{port} 查看文档
```
